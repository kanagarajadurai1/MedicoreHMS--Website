## 📌 Overview

**MediCore HMS** is a **single-page hospital management system** designed to demonstrate modern UI/UX and efficient frontend architecture using **vanilla JavaScript**.

This project focuses on delivering a **smooth, responsive, and interactive dashboard experience** without relying on any external frameworks or libraries.

## ✨ Features

- 🔐 Simple login interface (client-side validation)
- 🎨 Dark / Light mode toggle
- 📊 Dashboard with real-time stats UI
- 👤 Patient management interface (UI-based)
- 🩺 Doctor overview section
- 📅 Appointment display section
- 💳 Billing overview UI
- 🏥 Ward & bed visualization (basic)
- 🔔 Notification UI & alerts
- 📱 Fully responsive design (mobile + desktop)
